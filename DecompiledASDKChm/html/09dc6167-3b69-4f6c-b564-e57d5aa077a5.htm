Web service programming is supported in the ASDK through the various Web service layers. For programmatic access to ASDK Web services, programmers should create proxy classes to the Web services they want to use and include these proxy classes in their code. Alternatively, a user can access an ASDK Web service directly from a browser by adding to the Web service URL a query string representing a command and arguments. Code examples in this section are provided in C#. This section contains the following documentation:

  • Web Service URLs Reference
  • Creating and Using Proxy Classes
  • Setting Up Credentials
  • Using a Query String
