The SoftwareCommandLineDetails type exposes the following members.

Fields

  NameDescription
CommandLine
Command line itself.
CommandLineType
Type of command line. Following types are available: Custom, Install, Uninstall, Repair NOTE: the command line above owned and installed by Software Management Framework. Other solutions can expand functionality and import another command line types
Description
The brief description of the SoftwareCommandLineResource
FailureCodes
Comma separated list of failure codes.
Guid
Guid of SoftwareCommandLineResource
IsDefault
Makes this command line the default for the type of action that is selected in Command type. For example, you can define multiple installation command lines for a software resource, but only one installation command line can be the default.
ItemAttributes
Item attributes of SoftwareCommandLineResource
Name
The name of the SoftwareCommandLineResource
SoftwareComponentResource
Guid of SoftwareComponentResource that is associated with this SoftwareCommandLineResource.
SoftwarePackageGuid
Guid of SoftwarePackageResource that is associated with this SoftwareCommandLineResource. If GUID is empty then command line does not require package.
SuccessCodes
Comma separated list of success codes.
