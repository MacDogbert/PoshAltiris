Each unit of work that you can perform with the Deployment Solution ASDK is represented by a different XML string that is passed into the main API, RunDeploymentSolutionTask(String). The work to be done is defined by a unique combination of values in the <ItemName/> element and the <ItemAction/> element. The <ItemName/> element names a particular scope of work within the Deployment Solution, such as "Capture Personality," while <ItemAction/> identifies a particular kind of action to be performed, such as "Create," "Get," "GetAll," etc. (Each <ItemName/> only suports a specific subset of actions. For a comprehensive list of all supported item-action combinations, see this page.)
