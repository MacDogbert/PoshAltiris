DSH_ImageFileResource Class  Example  See Also  Send Feedback
Creates an XML string for use as an input to RunDeploymentSolutionTask. The values of this class's properties are inserted into the XML in the appropriate places.

Namespace:  Altiris.ASDK.DeploymentSolution.COM
Assembly:  Altiris.ASDK.DS.COM (in Altiris.ASDK.DS.COM.dll)

Syntax

C#
public string ToString(
	string action
)

Parameters

action
Type: System..::.String
The type of action to be performed. Valid actions for this method depend on the <ItemName/> and <ItemType/> elements of the XML input. See the Deployment Solution COM Programming page to find the specific action type(s) supported for the unit of work you want to perform.

Return Value

A string containing the XML that can be used as input to RunDeploymentSolutionTask.

Remarks

This "input helper" method is intended to facilitate the creation of the sometimes complex XML string that is used as input to the main API of the Deployment Soultion ASDK, RunDeploymentSolutionTask. While this class is intended for use principally with COM compatible scripts such as VBScript, it can be called from .NET code as well. For example, it can be used in C# code that calls into the Deployment Solution ASDK web service.

Examples

CopyVBScript
'----------------------------------------------------
' The following code is for "get" Action type
'----------------------------------------------------
set objImageFileResource = CreateObject( "Altiris.ASDK.DeploymentSolution.DSH_ImageFileResource" )
objImageFileResource.ImageName = "Windows2k8DiskImage.gho"

'----------------------------------------------------
' Get the input XML string.
'----------------------------------------------------
inputXml = objImageFileResource.ToString( "get" )
set dsDetails = dsManagement.RunDeploymentSolutionTask( inputXml )
