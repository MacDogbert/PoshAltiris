Example Scenarios

To view example scenarios and scripts that show how to use the ASDK APIs, navigate to your ASDK installation directory (typically, C:\Program Files\Altiris\Altiris ASDK) and open the Samples folder. Under this folder are a number of VBScript samples that demonstrate how to use nearly all the available ASDK API's in VBScript. You will also find a selection of COM (C#) and CLI samples for certain product components.

Note:

The VBScript examples are working VBScript applications. If you only want to view the scripting code, open the *.vbs files in Edit mode, or open them with Notepad or other text editor. Simply double-clicking these files will cause the scripts to execute.
