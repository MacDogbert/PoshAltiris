Parses the XML string stored in the DSDetails.ReturnXML property after a call to RunDeploymentSolutionTask, and stores the parsed values in the corresponding properties of this method's class.

Namespace:  Altiris.ASDK.DeploymentSolution.COM

Syntax

C#
public void Deserialize(
	Object dsDetails,
	string action
)

Parameters

dsDetails
Type: System..::.Object
The DSDetails object whose ReturnXML property contains the XML to be parsed, following a call to RunDeploymentSolutionTask.
action
Type: System..::.String
The action type. This must match the action type given to the previous call to RunDeploymentSolutionTask which returned the XML that is to be parsed.

Remarks

This "output helper" method is intended to facilitate the retrieval of data from the sometimes complex XML string that is returned by a call to the main API of the Deployment Solution ASDK, RunDeploymentSolutionTask. While this method is intended for use principally with COM compatible scripts such as VBScript, it can be called from .NET code as well. For example, it can be used in C# code that calls into the Deployment Solution ASDK web service.

To find the specific combinations of Item Name and Action for which this this method should be used, find all the rows in the tables on the Deployment Solution COM Programming page that list this method's class name under the Output Helper Class column.
