Generating GUIDs
Send Feedback

A GUID (Globally Unique Identifier) is a pseudo-randomly-generated number, mathematically guaranteed to be unique. GUIDS are used in software applications as unique identification. They are represented as 32-character strings that look like the following:

{199738f6-4562-4936-9445-dde153e9cfa9}

GUIDS can be generated using the uuidgen utility. Notification Server uses GUIDS to uniquely identify objects. GUIDS are used to identify the types of objects you can create in Notification Server (such as a collection), as well as to identify individual instances of an object (the "Development Computers" collection).

Several operations in the ASDK require you to provide the GUID (as a string) of the object targeted by an operation. If you created the object in your code, the GUID is generated for you and functionality is built in to retrieve it as a property of the object. To find the GUID for an existing Notification Server item, complete the following three steps:

  1. Open the Symantec Management Console.
  2. Right-click an object and select Properties.
  3. The GUID is displayed at the top of the Properties page and can be copied and pasted directly into your code.
